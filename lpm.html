4  El modelo de probabilidad lineal

4.1 Introducción

Eventos cuantitativos

¿Es posible usar el modelo de regresión para analizar los determinantes de eventos cuantitativos?

Variable dependiente binaria

La variable dependiente sólo toma dos valores:

  • y_i = 1, cuando se produce el evento que estamos estudiando;

  • y_i = 0, cuando no se produce.

Modelo de regresión múltiple

  • Modelo de regresión: y = \beta_0 + \beta_1 x_1 + \dots + \beta_k x_k + u.

  • ¿Cuál es la interpretación de los parámetros cuando y es una variable binaria?

Probabilidad condicional

  • Si y es una variable binaria, su esperanza condicional coincide con la probabilidad condicional de que y = 1: \mathop{\mathrm{E}}(y | \boldsymbol{x}) = \Pr(y = 1 | \boldsymbol{x}).

  • Modelo de probabilidad lineal (MLP): la probabilidad de que y tome el valor 1 es una función lineal en los parámetros \beta_0, \beta_1, \dots, \beta_k: \Pr(y = 1 | \boldsymbol{x}) = \beta_0 + \beta_1 x_1 + \dots + \beta_k x_k.

Interpretación de los parámetros

  • Podemos expresar el MLP como: p(\boldsymbol{x}) = \beta_0 + \beta_1 x_1 + \dots + \beta_k x_k, donde p(\boldsymbol{x}) es una forma abreviada de escribir \Pr(y = 1 | \boldsymbol{x}).

  • La pendiente \beta_j mide cómo cambia la probabilidad de que y = 1 cuando cambia x_j, manteniendo inalteradas el resto de explicativas: \beta_j = \frac{\Delta p(\boldsymbol{x})}{\Delta x_j}.

Ventajas y limitaciones

  • En comparación a otros modelos más sofisticados, el MLP:

    • es más fácil de usar e interpretar;

    • proporciona resultados similares cuando las variables explicativas toman valores cercanos a sus medias.

  • El MLP puede generar probabilidades fuera del intervalo (0, 1), por lo que no es adecuado para algunas aplicaciones.

4.2 Estimación por MCO

Estimación

  • Los parámetros del MLP se pueden estimar por MCO.

  • Los residuos se obtienen de la forma usual: \hat{u}_i = y_i - \hat{\beta}_0 - \hat{\beta}_1 x_{1i} - \dots - \hat{\beta}_k x_{ki}.

Predicción

  • Las predicciones de MCO se pueden interpretar como estimaciones de la probabilidad de y = 1: \hat{p}_i = \hat{\beta}_0 + \hat{\beta}_1 x_{1i} + \dots + \hat{\beta}_k x_{ki}

  • Para predecir la variable dependiente se suele usar la regla: \tilde{y}_{i} = \begin{cases} 1 & \text{si $\hat{p}_{i} \geq 0.5$}, \\ 0 & \text{si $\hat{p}_{i} < 0.5$}. \end{cases}

Bondad del ajuste

Tabulación cruzada de y e \tilde{y}:

\tilde{y}_i = 0 \tilde{y}_i = 1
y_i = 0 n_{00} n_{01}
y_i = 1 n_{10} n_{11}

  • Número de predicciones correctas: n_{00} + n_{11}.

  • Número total de observaciones: n = n_{00} + n_{01} + n_{10} + n_{11}.

  • Porcentaje de predicciones correctas: 100 (n_{00} + n_{11}) / n. Se usa para medir la bondad del ajuste de modelos con variable dependiente binaria.

Insesgadez y consistencia

  • Los supuestos RLM.1 a RLM.4 no son incompatibles con que la variable dependiente sea binaria.

  • Si se cumplen los supuestos RLM.1 a RLM.4 el estimador MCO es insesgado y consistente.

Heteroscedasticidad

  • En el modelo lineal de probabilidad se incumple el supuesto RLM.5.

  • La varianza condicional del término de error puede escribirse como: \mathop{\mathrm{var}}(u|\boldsymbol{x}) = p(\boldsymbol{x})\big(1 - p(\boldsymbol{x})\big).

Uso de MCO

  • Los parámetros del modelo lineal de probabilidad pueden estimarse por MCO.

  • Para contrastar hipótesis o construir intervalos de confianza es necesario usar estimaciones robustas a heteroscedasticidad de \mathop{\mathrm{var}}(\hat{\beta}).

4.3 Estimación eficiente

Varianza del término de error

  • La varianza condicional del término de error del MLP puede escribirse como: \mathop{\mathrm{var}}(u_i|\boldsymbol{x_i}) = p(\boldsymbol{x_i})\big(1 - p(\boldsymbol{x_i})\big).

  • La probabilidad p_i = p(\boldsymbol{x_i}) depende de parámetros desconocidos.

  • Se podría aplicar MCPF si se reemplaza p_i por una estimación, \hat{p}_i.

Mínimos Cuadrados Ponderados Factibles

  • Si se cumplen los supuestos RLM.1 a RLM.4, MCO es un estimador insesgado.

  • Con las estimaciones \hat{\beta}_0, \hat{\beta}_1, \dots, \hat{\beta}_k se estima la probabilidad de que y = 1: \hat{p}_i = \hat{\beta}_0 + \hat{\beta}_1 x_{1i} + \dots + \hat{\beta}_k x_{ki}

  • Los pesos para obtener el estimador MCPF son w_i = 1 / \hat{h}_i, donde: \hat{h}_i = \hat{p}_i (1 - \hat{p}_i).

Limitaciones

  • El estimador MCPF no puede obtenerse si para algunas observaciones se obtiene \hat{p}_i \leq 0 o \hat{p}_i \geq 1.

  • Aunque se han propuesto alternativas, en esos casos es mejor usar MCO y contrastes de hipótesis robustos a heteroscedasticidad.
